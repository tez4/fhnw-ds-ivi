## Install Repository Dependencies

``` bash
pip install poetry
```

``` bash
poetry lock --no-update
```

``` bash
poetry install --no-root
```

``` bash
poetry shell
```

To get the current environment name use:

``` bash
poetry env info -p
```

Install packages to environment:

``` bash
poetry add numpy
```
